# twitter-follower-remover

A simple Chrome extension for removing unwanted followers on X (Twitter).

Paste a list of usernames, click Start, and the extension will handle the rest
with safe delays and daily limits.

No external services. No data collection.

Short description...

## Usage
1. Open your followers page on X
2. Open the extension
3. Paste usernames **one per line**
4. Click Start
